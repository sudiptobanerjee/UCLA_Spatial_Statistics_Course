  1. Get county map for California from R “maps” library and create an “sf” object.
ca.county = map("county","california", fill=TRUE, plot=FALSE)
ca.county.sf <- st_as_sf(ca.county)
  1. Turn off the spherical geometry option as the “maps” library uses planar projections.
sf_use_s2(FALSE)
## Spherical geometry (s2) switched off
  1. Plot the county map of California
ca.coords = st_coordinates(st_centroid(ca.county.sf))
## Warning: st_centroid assumes attributes are constant over geometries
## Warning in st_centroid.sfc(st_geometry(x), of_largest_polygon =
## of_largest_polygon): st_centroid does not give correct centroids for
## longitude/latitude data
plot(st_geometry(ca.county.sf), main="County map of California")

  1. Add the county names.
plot(st_geometry(ca.county.sf), main="County map of California")
county.ID <- sapply(strsplit(ca.county$names, ","), function(x) x[2])
text(ca.coords, county.ID, cex=0.5, col = "red")

  1. Find the neighbors of Alameda county using tools in “spdep” library. We first create an “nb” object from the sf object. Next, we extract region id for the counties from the “nb” object. We use a bit of string splitting to extract thee name of the county only, create an adjacency matrix and finally extract the names of the neighbors from the adjacency matrix.
library(spdep)
## Loading required package: spData
## To access larger datasets in this package, install the spDataLarge
## package with: `install.packages('spDataLarge',
## repos='https://nowosad.github.io/drat/', type='source')`
ca.nb = poly2nb(ca.county.sf)
## although coordinates are longitude/latitude, st_intersects assumes that they
## are planar
ca.region.id = attr(ca.nb, "region.id")
ca.region.id = sapply(strsplit(ca.region.id, ","), function(x) x[2])
alameda.neighbors.index = ca.nb[[match("alameda", ca.region.id)]]
ca.adj.mat = nb2mat(ca.nb, style="B")
alameda.neighbors = rownames(ca.adj.mat[alameda.neighbors.index,])
alameda.neighbors = sapply(strsplit(alameda.neighbors, ","), function(x) x[2])
alameda.neighbors
## [1] "contra costa" "san joaquin"  "santa clara"  "stanislaus"
